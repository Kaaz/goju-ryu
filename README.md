### Zie je een foutje of heb je een idee?

Super! Dit is een project van en voor karateka's. Als je iets ziet dat niet klopt, een vertaling wilt verbeteren of een suggestie hebt, laat het vooral weten! Dat kan op twee manieren:

1.  Maak een **'Issue'** aan op deze GitHub-pagina om de fout of je idee te beschrijven.
2.  Of, als je handig bent, maak een **'Pull Request'** met de aanpassing.

Alle hulp wordt enorm gewaardeerd!
